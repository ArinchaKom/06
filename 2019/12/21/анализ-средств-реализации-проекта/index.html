Выбор языка реализации

Python

Python - это высокоуровневый язык программирования общего назначения, который ориентируется на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен и имеет удобную структуру, но в тоже время стандартная библиотека языка позволяет применять большой объём полезных функций и возможностей. Язык может использоваться для разработки веб-приложений, а так же написания прикладных приложений. Python поддерживает широкий перечень стилей разработки приложений: структурное, императивное, объектно-ориентированное, аспектно-ориентированное, функциональное, программирование. Основные характерные черты - динамическая типизация, автоматическое управление памятью, механизм обработки исключений, полная интроспекция, поддержка многопоточных вычислений, высокоуровневые структуры данных. Поддерживается разбиение программ на модули, а они в свою очередь, могут объединяться в пакеты. Одной из самых популярных интерпретаторов является CPython, написанный на С, поддерживающий большинство активных используемых платформ. Среда разработки распространяется бесплатно по свободной лицензии Python Software Foundation License, которая позволяет использовать его без ограничений в любых приложениях. Есть реализация интерпретатора для JVM с возможностью компиляции, CLR, LLVM, другие независимые реализации. Проект PyPy использует JIT-компиляцию, которая значительно увеличивает скорость выполнения Python-программ. Python активно развивается, новые версии с обновлениями выходят примерно раз в два года. Особенность языка - это отсутствие таких стандартов кодировки как ANSI, ISO, роль стандарта де-факто выполняет CPython, разрабатываемый под контролем автора языка. Python относится к наиболее востребованным и популярным языкам программирования, о чем свидетельствуют многочисленные рейтинги и анализ предложений на рынке разработки программных продуктов. Иногда Python сравнивают с такими популярными платформами как Ruby, но в отличии от него, Python требует меньше оперативной памяти, быстрее взаимодействует с процессором .
